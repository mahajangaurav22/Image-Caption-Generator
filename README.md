## Description
This project aims to develop an image captioning system that can automatically generate descriptive captions for uploaded images. The system uses a model that combines a Vision Transformer (ViT) as the encoder to process the image and a GPT -2-based decoder to generate the captions.

## Installation

<ol type="1">
  <li>Clone the repository: git clone https://github.com/mahajangaurav22/Image-Caption-Generator.git
    <li>Download the model from google drive link given above</li>
    <li>Install all the libraries mentioned in requirement.txt file
      <li>Run the web2.py file using the command -> streamlit run {path to web2.py file}
        </ol>
        
## Usage

<ul>
  <li>Use the upload file option on the left pane of the website to upload your desired photos
    <li> After the upload is complete, a generate caption button will appear.
      <li> Click on the button and wait for 2-3 seconds, your caption will be generated.
        
          </ul>
